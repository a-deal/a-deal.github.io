The Underlying Pattern

The Marine Corps doesn't use rigid control or no control. It uses elastic constraints: commander's intent pulls units toward alignment while allowing autonomous execution. That's the AI supervision model in one image.

LearnAIR's current curriculum trains line cooks (follow prompts, use tools). Mike's framework trains chefs (set intent, design systems, evaluate outputs). The April 7 pitch needs to make this distinction without insulting what LearnAIR already does. Frame it as: "You've built the foundation. We're proposing the next floor."

Optimization ratchet warning: If LearnAIR optimizes for "get veterans jobs fast" (Trisha confirmed revenue is the primary driver), they'll build a placement pipeline that measures speed and volume. That caps the program at tactical. The tension between revenue motivation and program quality is the load-bearing risk.
