00 What It Does
  • Records audio from your microphone during meetings
  • Transcribes using Whisper, running locally on Apple Silicon
  • Saves recordings and transcripts as plain files you own
  • Works offline. No accounts, no cloud, no subscriptions.

How It Compares

meet: Free, local, exportable transcripts, works offline, total privacy.

Granola: Free tier but no transcript export. Cloud-dependent.

Notion AI: $240/yr. Cloud-dependent.

01 Requirements
  • macOS (Apple Silicon recommended for fast transcription)
  • Homebrew installed. If not: brew.sh
  • Python 3
02 Install

Open Terminal and run:

git clone https://github.com/a-deal/meet.git
cd meet
./install.sh

The install script will:

  • Install sox (audio recording) via Homebrew if needed
  • Install mlx-whisper (transcription) via pip if needed
  • Add meet to your PATH
After install, restart your terminal or run source ~/.zshrc so the meet command is available.
03 Usage

Start recording:

meet start call-with-mike

Give it a name so you can find it later. Recording starts immediately.

Stop recording and transcribe:

meet stop

This stops the recording and automatically runs Whisper to generate a transcript. Transcription takes a minute or two depending on the length.

Transcribe an existing file:

meet transcribe ~/Downloads/some-recording.m4a

Works with any audio format (wav, m4a, mp3, etc.). ffmpeg handles the conversion.

List recordings:

meet list
04 Where Files Go
  • Recordings: ~/src/hub/meetings/recordings/
  • Transcripts: ~/src/hub/meetings/recordings/transcripts/

You can change the directory by setting the MEET_DIR environment variable:

export MEET_DIR="$HOME/meetings"
05 Tips
  • Use a good microphone. Built-in MacBook mic works, but an external mic gives cleaner transcriptions.
  • For Google Meet or Zoom calls, BlackHole (virtual audio driver) lets you capture both sides of the conversation. Install with brew install blackhole-2ch and set up an aggregate audio device in Audio MIDI Setup.
  • First transcription downloads the Whisper model (~1.5GB). After that it is cached.
  • Apple Silicon Macs transcribe roughly 10x faster than real-time. A 60-min meeting takes about 6 minutes.
>_ Using with Claude Code

After a meeting, you can feed the transcript to Claude Code for instant summaries, action items, or follow-up drafts:

cat ~/src/hub/meetings/recordings/transcripts/2026-03-22-call-with-mike.txt | claude "Summarize this meeting. List action items."

Or just tell Claude Code: "Read my latest meeting transcript and give me the key takeaways."
